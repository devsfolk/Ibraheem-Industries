### Resolution Requirements:
- **Hero Images**: 1920x1080px (Full HD landscape)
- **Product Cards**: 600x400px (3:2 ratio)
- **Product Detail Pages**: 800x600px (4:3 ratio)
- **CEO Portrait**: 400x400px (Square)
- **Section Images**: 800x600px or 1200x800px (Landscape)

### File Format:
- **Preferred**: JPG (for photos, optimized for web)
- **Alternative**: WebP (for better compression)
- **Avoid**: PNG (unnecessarily large file sizes for photos)

### Quality:
- **Compression**: 80-85% quality (balance between size and visual quality)
- **Color Space**: sRGB
- **Max File Size**: 200KB per image (optimize for web performance)
